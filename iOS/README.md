# nend-WebView Sample Project

![ロゴ](./WebViewAdSample/WebViewAdSample/Assets.xcassets/AppIcon.appiconset/Icon-App-83.5x83.5@2x.png)

## Overview
iOSアプリで nend のバナー広告を UIWebView で表示するサンプルプロジェクトです。
基本的な実装方法および動作の確認が行えます。  

## Requirement
* **Xcode7.0**以上
* iOS7以上
