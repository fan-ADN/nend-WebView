# nend-WebView Sample Project

![logo](https://user-images.githubusercontent.com/9563381/31269103-17daebce-aaba-11e7-9899-742435c4ef20.png)

## Overview
iOSアプリで nend のバナー広告を UIWebView で表示するサンプルプロジェクトです。
基本的な実装方法および動作の確認が行えます。  

## Requirement
* **Xcode7.0**以上
* iOS7以上
